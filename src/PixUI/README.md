# 工程目录结构
```shell
Animation             动画

Core                  通用的数据结构与枚举

Input                 键盘/鼠标等输入

Platform              平台相关
|----Common           各端通用的数据结构
|----Native           除Web端的原生实现
|----Web              Web端的虚拟实现

Route                 路由

State                 状态管理

Widgets               内置常用UI组件
|----Overlay          在Overlay层绘制的一些装饰器及弹窗组件
```
